# Introduction to HTML

## What is HTML?
HTML (HyperText Markup Language) is the standard language used to create web pages.
It gives structure and meaning to the content on the web.

## Key Concepts Learned
- HTML is a markup language, not a programming language
- Web pages are made up of HTML elements (tags)
- Each element usually has an **opening tag** `<tag>` and **closing tag** `</tag>`
- HTML files have the extension `.html`
- Browsers read HTML files and display the content

## Basic HTML Structure
```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>
```
## Explanation of the Structure

- `<!DOCTYPE html>` → Defines the document type
- `<html>` → Root element of the page
- `<head>` → Contains page metadata
- `<title>` → Title shown in the browser tab
- `<body>` → Visible content of the page

## Common HTML Tags

- Headings: `<h1>` to `<h6>`
- Paragraph: `<p>`
- Link: `<a href="url">text</a>`
- Image: `<img src="image.jpg" alt="description">`
- Line Break: `<br>`

```html
<!DOCTYPE html>
<html>
<head>
    <title>HTML Basics</title>
</head>
<body>
    <h1>Welcome to HTML Basics</h1>

    <h2>Headings</h2>
    <h1>Heading 1</h1>
    <h2>Heading 2</h2>
    <h3>Heading 3</h3>

    <h2>Paragraphs</h2>
    <p>This is a paragraph of text. HTML paragraphs are written inside &lt;p&gt; tags.</p>
    <p>This is another paragraph for demonstration.</p>

    <h2>Links</h2>
    <p>Visit <a href="https://www.example.com">Example Website</a> for more information.</p>

    <h2>Images</h2>
    <p>Below is an example image:</p>
    <img src="https://via.placeholder.com/400x150" alt="Sample Image">

    <h2>Line Breaks</h2>
    <p>This is the first line.<br>This is the second line using &lt;br&gt;.</p>
</body>
</html>
```

## Why HTML is Important

- Foundation of all websites
- Works with CSS and JavaScript to build functional web pages
- Easy to learn and widely used
